### 1. Intro <a name="intro"></a>

A basic "Movie Set" project that loads movie information and ratings from a data set, and produces a quick review of a selected movie.

### 1. Use Cases <a name="usecases"></a>

*	Load Information: load movie info files to memory.
*	Load Tags and Ratings: load movie tags \& ratings files to memory, for each movie.
*	Search Movies: find movies by name, id, actor, genre, producer name, origin country. 
*	Generate Review: produce a quick review of a selected movie.
